# Expense Tracking App

  

---

## 📌 Project Overview

In today's fast-paced world, individuals struggle to track daily expenses and manage finances effectively. Most people lack a clear picture of their spending habits, leading to overspending and poor financial planning.  

The **Expense Tracking App** is a user-friendly mobile application that allows users to:  

- Track daily expenses and income in real-time  
- Set monthly budgets for different expense categories  
- View transaction history and financial summaries  
- Receive warnings when approaching or exceeding budget limits  
- Manage personal financial data securely and privately  

---

## 🎯 Objectives

1. Develop a cross-platform mobile application using Flutter  
2. Implement user authentication with Firebase Authentication  
3. Store user data locally using SQLite for offline access  
4. Create an intuitive UI/UX with proper navigation and state management  
5. Implement budget management with real-time overspending alerts  
6. Provide comprehensive transaction history with filtering  

---

## 🛠 Technologies Used

**Frontend:**  
- Flutter  
- Dart  

**Backend & Database:**  
- Firebase Authentication (User login/signup)  
- SQLite (Local database for expenses, income, budgets)  

---

## ⚙️ System Working

1. **User Registration & Authentication** – Secure login/signup  
2. **Expense Management** – Add, edit, delete expenses  
3. **Income Tracking** – Add income with categories  
4. **Budget Setting** – Set monthly budgets and receive alerts  
5. **Transaction History** – View all transactions with filters  
6. **Profile Management** – Update profile, logout, or delete account
